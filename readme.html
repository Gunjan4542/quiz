Installation

Steps :
  1. Create a database and name it as "quiz" in phpmyadmin. You can give any name to this database. After that open "connection.php" file of this project and then edit the $server_name,$user_id,$password. and $database_name to desired values.
  2. Next , import two files "questions.sql" and "users.sql" located in "tables" directory of this project to your newly created "quiz" database
  3. Now You are done. go to "admin.php" page to open admin panel where you can add questions. Please add atleast 10 question using admin panel otherwise you will see an error called "undefined index" in the main contest page because that contest is programmed in a such way that it has 10 questions.You can edit number of question in "quiz.php" file.Also you can set number of attempts to prevent further login using the "attempt" field of "users" table.
  4. By default the contest time is 10 min. You can change this time. To change the contest duration, edit the variable m in "scripts.js" file to desired time in minutes. Also find the variable time_in_second in "quiz.php" file and add the total contest time in seconds.Clear your browser cookies and cache after this and Enjoy!!!
